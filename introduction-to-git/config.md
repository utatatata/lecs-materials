# Gitの設定

## グローバルとローカル

Gitの設定にはグローバル(常に有効)とグローバル(プロジェクト内でだけ有効)があります.

- `~/.gitconfig` グローバルの設定
- `./.git/config` ローカルの設定

プロジェクト内ではローカルの設定が優先されます.


## 設定の方法

グローバルの設定ファイルは `~/.gitconfig` ですが, なければ作成してください.

ローカルの設定ファイルは `git init` によって自動的に作成されます.

## 基本的な設定 (推奨)

`~/.gitconfig` (グローバル)

```
# ユーザに関する設定
[user]
  # コミットログなどに残る
  email = yours@example.com
  name = your name

[merge]
```
