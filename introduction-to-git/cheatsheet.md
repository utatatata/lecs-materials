# Gitチートシート

## ローカル作業の流れ

### ステージング

```
# すべてステージング
$ git add -A

# ./src 以下のファイルをすべてステージング
$ git add ./src

# ./test 以下の *.hs (ワイルドカード)をすべてステージング
# ディレクトリを省略した場合 (*.hs) は, カレントディレクトリを指定したこと (./*.hs) になる
$ git add ./test/*.hs
```

### コミット

```
# デフォルトのエディタが起動し, コミットメッセージを編集できる
$ git commit

# メッセージを指定できるが基本1行しか書けない (非推奨)
$ git commit -m "a commit message"
```
